# codeigniter-posty

Docker Tutorials for CodeIgniter:

1. PHP CodeIgniter 4 + PostgreSQL + NGINX
```
Requirements:
- CodeIgniter 4.2.5
- PostgreSQL 14.5
- PHP: fpm-alpine3.16
- NGINX: stable-alpine
```
2. PHP CodeIgniter 4 + MS. SQL SERVER + NGINX
```
- CodeIgniter 4.2.5
- Ms. SQL Server 17
- PHP: 7.4.30-fpm
- NGINX: stable-alpine
```

For more information, please visit [Docker Hub](https://hub.docker.com/)
